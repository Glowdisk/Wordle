## 2. Data Collection

By playing this game, the following data is automatically collected and transmitted to a third-party service (Google Forms) upon the start of each session and at the end of each game:

### 2.1 Network & Location Data (collected at launch and admittedly terribly inaccurate)
| Data | Description |
|------|-------------|
| Public IP Address | Your ISP-assigned IP address |
| City | Approximate city derived from your IP address |
| Region | Approximate region/state derived from your IP address |
| Country | Country derived from your IP address |
| ZIP Code | Approximate ZIP code derived from your IP address |
| Latitude | Approximate latitude derived from your IP address |
| Longitude | Approximate longitude derived from your IP address |
| ISP | Your Internet Service Provider name |

### 2.2 Gameplay Data (collected at end of game)
| Data | Description |
|------|-------------|
| Correct Word | The word that was selected for that session |
| Result | Whether the User guessed the word correctly |
| Guess Count | The number of guesses the User made |
